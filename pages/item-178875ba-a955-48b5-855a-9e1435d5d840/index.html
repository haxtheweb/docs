HAX pages are authored in HTML (including web components), and each page save now writes multiple machine-friendly formats in addition to the canonical page HTML.

Formats generated on every page save

  • .html - Canonical rendered page content
  • .md - Markdown version of the page content
  • .json - Structured page payload including metadata and content
  • .yaml - Structured page payload including metadata and content in YAML
  • .xml - Structured page payload including metadata and content in XML

For any page route, append an extension to access the alternate format directly. Example for this page: /documentation/data-formats.md, /documentation/data-formats.json, /documentation/data-formats.yaml, and /documentation/data-formats.xml.

Authoring and media formats supported in HAX

  • MathJAX
  • LaTeX
  • Markdown / md (convert to HTML or embed source as HTML)
  • DocX (convert to HTML)
  • Web Components / HTML
  • Images (png, jpg, jpeg)
  • Video (mp4)
  • Audio (mp3 file or browser-based recording)
