HAX11ty is a merger between HAX and 11ty. It uses these technologies in the following way:

  • User edits via HAX (future) or modifies files locally
  • hax11ty script watches for changes and rebuilds files w/ 11ty, refreshing the screen
  • this is compiled to a valid HAXcms data package to be used either completely static with a CDN or locally in HAXcms PHP version
